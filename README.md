## Overview

This repository contains phylogenetic analysis results of the Chikungunya Virus, visualizing transmission dynamics and evolutionary relationships across global populations.

## Repository Contents

- **`auspice/`** - Contains Nextstrain JSON files for interactive phylogenetic tree visualization
- **`config/`** - Configuration and metadata files used in the analysis
- **`data/`** - Input datasets
- **`results/`** - Analysis outputs
- **`Snakefile`** - Workflow automation script

## Explore the Data

### How to Visualize the Global Dataset

Follow these simple steps to explore the interactive phylogenetic trees:

1. **Download the dataset file**
   - Go to the `auspice/` directory in this repository
   - Download the `Chikv_Asia.json` file for the Asian CHIKV genomes and the `chikv_global.json` file for the global view

2. **Open the Auspice visualization platform**
   - Visit https://auspice.us in your web browser

3. **Load your data**
   - Drag and drop or upload the `Chikv_Asia.json` file or the `chikv_global.json` onto the web page

4. **Explore the visualization**
   - The interactive phylogenetic tree will load immediately
   - You can now explore transmission patterns, evolutionary relationships, and geographic distribution of Chikungunya Virus globally or in Asia
