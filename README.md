# ChIKV_Global
Transmission dynamics of Chikungunya Virus around the world

### How to visualise the global dataset

  1. Download the chikv_modified.json file in the auspice directory

  2. Navigate to https://auspice.us in your web browser

  3. Drag and drop your JSON file onto the web page

  4. The visualization will load immediately, allowing you to explore your data
